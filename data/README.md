# Data

Subject instance images are sourced from the official **DreamBooth dataset** ([google/dreambooth](https://github.com/google/dreambooth)) and stored in `artifacts/subject_images/` via Git LFS. We use three subjects: `dog`, `dog2`, and `backpack` (5 images each).

Class regularization images (100 per subject) are generated by the unmodified SD 1.5 model before training and stored in `artifacts/class_images/`. To regenerate them:

```bash
python code/main.py generate --config code/configs/generate.yaml
```

To download the subject images directly from the DreamBooth dataset:

```bash
pip install huggingface_hub
python -c "
from huggingface_hub import snapshot_download
snapshot_download('google/dreambooth', repo_type='dataset', local_dir='dreambooth_dataset')
"
```
